## About

Roundtable is an open-source framework that lets LLMs play different thinking personas, cross-challenge each other, and reach a structured consensus. It ships with 21 built-in persona recipes (Kahneman, Munger, Inamori, Jobs, …) covering cognitive bias, inversion thinking, Eastern management philosophy, and mind cultivation.
